第④层 · 最锋利的一刀

「不受就业歧视」是怎么判定的

性别歧视在招聘里不是均匀分布的,它集中在特定岗位结构。判一个专业友不友好,看它的主要就业出口属于哪一类:

⭐ 歧视最低

考公 / 考编 / 国企 / 事业单位 / 医院 / 学校。考试录取,法律禁止性别歧视,且女性常占多数。

电气(国网)· 师范 · 医学技术 · 计算机考公
中等

市场化技术岗(写代码/算法/测试)。有隐性偏好,但相对看能力,女生站得住。

计算机 · 软件 · 数据科学
⛔ 歧视最高

施工 / 野外 / 现场 / 倒班。明确"限男",女生基本进不去核心岗。

土木 · 采矿 · 地质 · 机械现场

→ 对双非女生,"不受歧视"最稳的打法 = 选十五五硬科技方向里,同时能走"体制内/国企/考编"出口的专业:① 蹭政策红利;② 招聘按考试走,女生不被卡;③ 双非也有公平入场券。这是整份推荐的灵魂。
